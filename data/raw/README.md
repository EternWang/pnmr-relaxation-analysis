# Raw data provenance

- `DS0001A.CSV` is the exported Meiboom-Gill echo-train waveform used for the quantitative `T2` fit.
- `t1_inversion_recovery.csv` is the hand-transcribed inversion-recovery table used for the `T1` analysis.
- `t1_zero_crossing.json` stores the independently measured zero-crossing delay used for the primary `T1` estimate.
- `part_a_cursor_measurements.csv` records the cursor-based pulse-programmer checks from the apparatus setup stage.
